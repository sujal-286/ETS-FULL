⚙ Setup Guide — Google Drive + Sheets

Step 1 — Sheet structure

Create a Google Sheet with this exact header row:

Name | Latitude | Longitude | DateTime | Category | Notes | Images | RowID

The Images column stores Google Drive direct URLs (comma or newline separated). The Apps Script fills this automatically.

Step 2 — Create a Google Drive folder

In Google Drive, create a folder called MapTrack Photos. Right-click → Share → Anyone with link → Viewer. Copy the folder ID from the URL.

Step 3 — Apps Script (handles write + Drive upload)

In your Sheet: Extensions → Apps Script → paste this script → Deploy → Web App → Execute as Me → Anyone → copy URL.

// ── CHANGE THIS to your Drive folder ID ──
const FOLDER_ID = 'YOUR_DRIVE_FOLDER_ID';

function doPost(e) {
  var ss  = SpreadsheetApp.getActiveSpreadsheet();
  var sh  = ss.getActiveSheet();
  var data = JSON.parse(e.postData.contents);
  var action = data.action;

  if (action === 'append') {
    sh.appendRow([data.name, data.lat, data.lng,
      data.datetime, data.category, data.notes,
      data.images, data.rowId]);

  } else if (action === 'update') {
    var rows = sh.getDataRange().getValues();
    for (var i=1; i<rows.length; i++) {
      if (String(rows[i][7])===String(data.rowId)) {
        sh.getRange(i+1,1,1,8).setValues([[
          data.name, data.lat, data.lng,
          data.datetime, data.category,
          data.notes, data.images, data.rowId
        ]]);
        break;
      }
    }
  } else if (action === 'delete') {
    var rows = sh.getDataRange().getValues();
    for (var i=1; i<rows.length; i++) {
      if (String(rows[i][7])===String(data.rowId)) {
        sh.deleteRow(i+1); break;
      }
    }
  } else if (action === 'uploadImage') {
    // Upload base64 image → Google Drive
    // Returns direct view URL
    var folder = DriveApp.getFolderById(FOLDER_ID);
    var blob = Utilities.newBlob(
      Utilities.base64Decode(data.imageBase64),
      'image/jpeg',
      data.filename || ('photo_' + Date.now() + '.jpg')
    );
    var file = folder.createFile(blob);
    file.setSharing(
      DriveApp.Access.ANYONE_WITH_LINK,
      DriveApp.Permission.VIEW
    );
    var fileId  = file.getId();
    var viewUrl = 'https://drive.google.com/uc?export=view&id=' + fileId;
    return ContentService
      .createTextOutput(JSON.stringify({ status:'ok', url: viewUrl, fileId: fileId }))
      .setMimeType(ContentService.MimeType.JSON);
  }

  return ContentService
    .createTextOutput(JSON.stringify({ status:'ok' }))
    .setMimeType(ContentService.MimeType.JSON);
}

function doGet(e) {
  return ContentService.createTextOutput('OK');
}
Step 4 — Publish Sheet for reading

File → Share → Publish to web → CSV → Publish → copy URL → paste into ★ Published CSV URL field in the Sheet tab.

Step 5 — ESP32 image upload flow

Your ESP32-CAM sends the photo as base64 to the Apps Script URL with action: "uploadImage". The script saves it to Drive and returns the direct URL. Then a second call with action: "append" saves the URL + GPS to the Sheet. The code I gave you earlier handles this automatically.
